Liquid Water

The following script can be used to build a simulation box with 500 water molecules and density of 1 g/cm³. In the end, it creates a LAMMPS data file that can be used to run an MD simulation. By changing the value defined in the first command, one can choose among various 3-site water models.

1 define model as TIP3P
2 define density as 0.998 # g/cm³
3 define N as 500
4 include $model.playmol
5 atom_type HW $LJ_H
6 atom_type OW $LJ_O
7 mass HW 1.008
8 mass OW 15.9994
9 bond_type HW OW $bond_OH
10 angle_type HW OW HW $angle_HOH
11 atom H1 HW $charge_H
12 atom O OW $charge_O
13 atom H2 HW $charge_H
14 bond O H1 H2
15 box density {0.602214*$density} # Da/Å³
16 packmol diameter 3.0 retry 0.95 pack mol(O) $N action execute
17 write lammps water_$model.lmp

A file that contains model parameters must conform to the LAMMPS model equations. In the case of harmonic potentials for bond stretching and angle bending, these equations are, respectively:

\[ V_{bond} = k_b \left(r - r_0\right)^2 \qquad \text{and} \qquad V_{angle} = k_a \left(\theta - \theta_0\right)^2 \]

The intermolecular potential is:

\[ V_{ij} = 4 \epsilon_{ij} \left[ \left(\frac{\sigma_{ij}}{r}\right)^{12} - \left(\frac{\sigma_{ij}}{r}\right)^6 \right] + C\frac{q_i q_j}{r} \]

where \(\epsilon_{ij} = \sqrt{\epsilon_i \epsilon_j}\) and \(\sigma_{ij} = (\sigma_i + \sigma_j)/2\). In the files shown below, the units of the parameters are in conformance with the LAMMPS unit style real.

Linear Hydrocarbons

The following script creates a simulation box for with single-component, united-atom, linear hydrocarbon molecules. The units are consistent with the LAMMPS unit style real.

1 define N as 5
2 define FF as nerd
3 define seed as 3467
4 define nmols as 500
5 define rho as 0.6 # g/cm³
6 define temp as 180 # K
7 
8 mass CH4 16.04242 # Da
9 mass CH3 15.03450 # Da
10 mass CH2 14.02658 # Da
11 
12 include $FF.playmol
13 
14 if {$N == 1} then
15  atom C1 CH4
16 else
17  atom C1 CH3
18  for i from 2 to {$N-1}
19  atom C$i CH2
20  bond C$i C{$i-1}
21  next
22  atom C$N CH3
23  bond C$N C{$N-1}
24 endif
25 
26 if {$N > 3} then
27  build
28  $N
29  C1 0 0 0
30  C2 C1 $L
31  C3 C2 $L C1 $theta
32  for i from 4 to $N
33  C$i C{$i-1} $L C{$i-2} $theta C{$i-3} $phi
34  next
35 endif
36 
37 box density {0.602214*$rho} # Da/Å³
38 align mol(C1) x y
39 packmol seed $seed retry 0.95 copy mol(C1) $nmols action execute
40 
41 write lmp/models C${N}_$FF.lammpsdata
42 write summary

File: nerd.playmol

The following file contains the NERD [4] model parameters:

1 define cutoff as 13.8 # Å
2 define L as 1.54 # Å
3 define theta as 114 # degrees
4 define phi as 180 # degrees
5 define kB as 1.987E-3 # kcal/mol.K
6 
7 diameter CH4 3.730 # Å
8 diameter CH3 3.910 # Å
9 diameter CH2 3.730 # Å
10 
11 atom_type CH2 lj/cut {45.8*$kB} 3.930 $cutoff
12 if {$N == 2} then
13  atom_type CH3 lj/cut {100.6*$kB} 3.825 $cutoff
14 else
15  if {$N == 3} then
16  atom_type CH3 lj/cut {102.6*$kB} 3.857 $cutoff
17  else
18  atom_type CH3 lj/cut {104.0*$kB} 3.910 $cutoff
19  endif
20 endif
21 atom_type CH4 lj/cut {148.0*$kB} 3.730 $cutoff # TraPPE parameters
22 
23 bond_type CH? CH? harmonic {96500*$kB/2} $L
24 angle_type CH? CH? CH? harmonic {62500*$kB/2} $theta
25 dihedral_type CH? CH? CH? CH? opls {355.04*2*$kB} {-68.19*2*$kB} {701.32*2*$kB} 0.0
