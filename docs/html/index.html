Introduction

Playmol is a(nother) software for building molecular models.

Its main distinguishing features are:

  • Molecules are created with simple scripts consisting of a small set of commands.
  • Molecular topology arises naturally when atoms are connected (automatic detection of angles and dihedrals).
  • Multiple copies of a molecule are automatically created when new coordinates are defined for their atoms.
  • Integration with Packmol [3] provides a way of creating complex molecular systems.
  • Generation of LAMMPS [5] configuration files provides a way of performing efficient MD simulations.

The sections of this manual are:

  • Installation: provides instructions for Playmol installation
  • Playmol Basics: explains the working logic of Playmol
  • Commands: describes all Playmol commands
  • Sample Scripts: provides sample Playmol scripts
